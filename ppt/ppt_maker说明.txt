本脚本会基于一定规则匹配并替换给定ppt模板中所有文本部分的{...}
{}格式为：{type data arg1 arg2 ...}，其中type为文本类型，data为文本内容，arg1 arg2 ...为可选参数，用于控制文本的显示格式，如百位分隔符，时间格式等。
type的取值范围如下：
    int 整数
        可选参数：, 是否使用百位分隔符
                + 大于等于零时是否显示正号
    float 浮点数
        可选参数：, 是否使用百位分隔符
                + 大于等于零时是否显示正号
                .x 保留小数位数，如.2表示保留两位小数
                .x% 百分比，如0.2%表示保留两位小数的百分数
    str 字符串
    pic 图片
        特殊格式，会删去文本并以指定参数插入图片
        data为文件路径（注意相对路径是相对于脚本文件的路径，路径中不能有空格）（文件路径中允许嵌套{}实现路径变量）
        必须参数  (x, y) 距左上角的水平和垂直距离，单位为厘米
                (w, h) 图片宽高，单位为厘米
    time 时间
        必须参数：format 时间格式，如%Y年%m月%d日，结果为2022年01月01日。%Y指年份（四位），%y指年份（两位），%m指月份（两位），%d指日期（两位），%H指小时（24小时制，两位），%M指分钟（两位），%S指秒（两位）
        （本年度第几周参数在这里不支持，在榜单信息类中单独给出一个int）
    colorint, colorfloat 带颜色的整数/浮点数
        在int/float类型的基础上，会根据数值的正负给出不同的颜色
        类型名后可加_$x 表示使用第x套配色方案，如signedint_$2，不加则使用第零套
        具体颜色在config_color中定义
        可用参数与上述对应类型相同
    Table 表格
        做出特定内容的表格，data目前只能为：
            top1view_recent_10
            total_view_recent_10
            startpt_recent_10
            side_startpt_recent10
        表示做出相应的表格
data的格式如下：
    除有特殊说明外，一般格式为x:y 表示x对象的y属性
        这里x对象分为两种：
            1. 榜单信息类，包括pres（当期）,prev（上期），history（历史回顾对应期）
                拥有的属性为：
                    index 期数（整数）
                    first 本期第一名,这会给出一个视频信息类对象，要调用信息需要按相应属性继续调用，如pres:first:view
                    start_time 收录开始时间（时间类型）
                    end_time 收录结束时间（时间类型）
                    week （结束日期在）本年度第几周（整数）
                    （以下信息均为整数，pres中的以下信息均可加_$delta表示与上期的差（整数），加_$rate表示变化率（浮点数），不单独写出）
                    new_total 新曲数量
                    ori_total 原创曲数量
                    view 总播放量
                    danmuku 总弹幕量
                    reply 总评论量
                    favorite 总收藏量
                    coin 总硬币量
                    share 总分享量
                    like 总点赞量
                    startpt 主榜起分
                    top1_view 最高播放量
                    side_startpt 副榜起分
                以下是国产榜限定属性，各引擎新曲数量
                    袅袅虚拟歌手
                    MUTA
                    Sharpkey
                    DeepVocal
                    AISingers
                    X Studio
                    VocalSharp
                    Vogen
                    跨引擎
            2. 视频信息类，如prev:first，表示上期第一名
                拥有的属性为；
                    aid av号（纯整数，不带av）
                    bvid bv号（不带bv）
                    title
                    mid
                    up主
                    投稿时间 （注意这里是按数据表中的格式给出字符串）
                    时长
                    分P数
                    播放增量
                    弹幕增量
                    评论增量
                    收藏增量
                    硬币增量
                    分享增量
                    点赞增量
                    Pt
                    修正A
                    修正B
                    修正C
                    Last Pt
                    rate
                    长期入榜及期数
                    新曲排名
                    staff
                    原创 （其他，原创或榜外原创）
                    引擎
                    （以上是数据表中所有表头，这些信息均可调用）
                    pub_time 投稿时间（时间类型）
